GetIP
=====

Узнает внешний IP-адрес и отправляет его на e-mail

================================================
Version 1.0.0
Работает по протоколу SMTP, порт 25, без шифрования SSL.

================================================
Version 1.0.1
Работает по протоколу SMTPS, порт 465, c шифрованием SSL.
